// Pisah tiap slide dengan baris "=== Slide ===". Baris "//" = catatan, diabaikan.
// SEMUA field opsional — kalau "layout" tidak ditulis, jenis slide dideteksi
// otomatis dari isinya (poin -> list/tabel, dua kolom -> komparasi, dst).

// ARTI TIAP FIELD:

// sub     : kalimat pendukung di bawah judul
// tombol  : tombol pil kuning di bawah teks
//
// LIST  : tiap poin diawali "-" atau emoji. Format "Judul :: penjelasan".
// TABEL : poin format "Nama :: Angka" (nomor urut otomatis).
// KOMPARASI (otomatis): pakai "kiri:" & "kanan:" + poinnya — ATAU cukup dua
//   baris judul kolom yang diakhiri ":" lalu poin di bawah masing-masing.
// MEME (otomatis): isi "captop:" / "capbawah:" (atau tulis "meme" di topik).
//   Gambar meme di-upload di editor setelah import.
