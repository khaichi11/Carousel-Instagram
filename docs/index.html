Aturan format brief (biar slide kebentuk otomatis)
  1. Pisah slide pakai baris === Slide === (atau header Slide 1 — Topik). Baris diawali // = catatan, diabaikan.
  2. Field per slide (semua opsional): layout = jenis slide (cover / highlight / list / table / compare / meme / figure / cta) · theme = warna dasar (dark = navy, light = terang) · topik = teks badge pil kuning di pojok kanan atas · eyebrow = label kecil kuning miring di atas judul (mis. "STOP DULU!") · judul = teks utama · sub = kalimat pendukung di bawah judul · tombol = tombol pil kuning. Kalau layout nggak ditulis, jenis slide dideteksi otomatis dari isi.
  3. Highlight kata penting: bungkus pakai **kata** → jadi stabilo kuning, ikut ke PNG & PPTX.
  4. List / Tabel: tiap poin diawali - atau emoji. Format list: Judul :: penjelasan. Tabel: Nama :: Angka (nomor urut otomatis).
  5. Komparasi — otomatis: tulis kiri: & kanan: diikuti poin masing-masing — atau cukup dua baris judul kolom yang diakhiri : lalu poinnya. Dua kolom langsung kebentuk.
  6. Meme — otomatis: isi captop: / capbawah: (atau tulis "meme" di topik). Gambar meme di-upload di editor setelah import.
  7. Aturan upload PDF: harus PDF teks — hasil Save/Export as PDF dari Google Docs, Word, atau Notion, dengan struktur sama kayak contoh di samping. PDF hasil scan/foto tidak bisa dibaca (teksnya bukan teks asli), jadi otomatisasinya gagal.

Generate with ChatGPT

Dua cara: isi form singkat, atau tempel naskah/hasil brainstorming kamu di editor naskah. Dua-duanya menghasilkan prompt siap paste ke ChatGPT.

⚡ Males isi satu-satu? Tempel format ==== topik: … di sini

Tempel satu blok teks berformat, klik Isi form dari teks, semua field kebawah keisi otomatis. Cocok kalau kamu minta ChatGPT nulisin brief-nya dulu dengan format ini.

Alur: Copy format buat ChatGPT → paste ke ChatGPT biar diisiin → hasilnya tempel balik ke kotak ini → Isi form dari teks → Generate Prompt → paste lagi ke ChatGPT buat naskah slide-nya.

Aturan format naskah (Bahasa Indonesia)

Ini format yang dimengerti tombol Buat Slide dari Teks. Prompt di atas sudah menyuruh ChatGPT menulis persis format ini — panduan berikut buat kamu yang mau nulis/edit manual.

  • Pisah tiap slide pakai baris === Slide ===. Baris diawali // = catatan, diabaikan.
  • layout: cover · highlight · list · table · compare · meme · figure · cta
  • theme: dark (navy) / light (terang) · align: top / center / bottom
  • topik: badge pil kuning kanan atas · eyebrow: label kecil miring di atas judul
  • judul: teks utama · sub: kalimat pendukung · tombol: tombol pil kuning
  • Stabilo: bungkus kata penting pakai **kata**
  • List: tiap poin diawali -, boleh Judul :: penjelasan · Tabel: Nama :: Angka
  • Komparasi: kiri: lalu poinnya, terus kanan: lalu poinnya
  • Meme: captop: dan capbawah: (gambarnya di-upload di editor slide)

ChatGPT akan menambah satu prompt gambar bahasa Inggris di paling bawah naskah (bukan per slide), sebagai baris // biar diabaikan pas build slide tapi tetap kebaca kalau di-copy. Memakai base style tetap di bawah + adegan sesuai tema carousel. Base style-nya bisa kamu ubah.
