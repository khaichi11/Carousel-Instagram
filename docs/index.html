Aturan format brief (biar slide kebentuk otomatis)
  1. Pisah slide pakai baris === Slide === (atau header Slide 1 — Topik). Baris diawali // = catatan, diabaikan.
  2. Field per slide (semua opsional): layout = jenis slide (cover / highlight / list / table / compare / meme / cta) · theme = warna dasar (dark = navy, light = terang) · topik = teks badge pil kuning di pojok kanan atas · eyebrow = label kecil kuning miring di atas judul (mis. "STOP DULU!") · judul = teks utama · sub = kalimat pendukung di bawah judul · tombol = tombol pil kuning. Kalau layout nggak ditulis, jenis slide dideteksi otomatis dari isi.
  3. Highlight kata penting: bungkus pakai **kata** → jadi stabilo kuning, ikut ke PNG & PPTX.
  4. List / Tabel: tiap poin diawali - atau emoji. Format list: Judul :: penjelasan. Tabel: Nama :: Angka (nomor urut otomatis).
  5. Komparasi — otomatis: tulis kiri: & kanan: diikuti poin masing-masing — atau cukup dua baris judul kolom yang diakhiri : lalu poinnya. Dua kolom langsung kebentuk.
  6. Meme — otomatis: isi captop: / capbawah: (atau tulis "meme" di topik). Gambar meme di-upload di editor setelah import.
  7. Aturan upload PDF: harus PDF teks — hasil Save/Export as PDF dari Google Docs, Word, atau Notion, dengan struktur sama kayak contoh di samping. PDF hasil scan/foto tidak bisa dibaca (teksnya bukan teks asli), jadi otomatisasinya gagal.
